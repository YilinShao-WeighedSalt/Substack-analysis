# Semi Doped: Qualcomm's HBC Memory, Alphawave, Modular, and more

Austin Lyons + Vik Sekar break down Qualcomm's NYC Investor Day. Qualcomm's message: phones are maturing; the future is data-center and automotive/edge. Goal: two-thirds of revenue non-handset by FY2029.

**High Bandwidth Compute (HBC):** stack cheaper high-capacity **LPDDR** memory directly *on top of* the processing logic (via a logic/accelerator die under the memory) instead of HBM towers beside it. Exposes the whole chip face for ~100,000 interconnects vs ~2,000 out the side — big bandwidth without TSMC CoWoS advanced packaging. Hosts note it resembles startup **d-Matrix's** in-memory compute; hard problems (thermals, planarity, TSV keep-out density loss when stacking multiple DRAM layers) are *moved*, not solved. First HBC chip = **AI250 in 2027** (AI200 in 2026 has no HBC; AI300 in 2028).

**"Disaggregated inference"** is the strategic opening — inference splits into prefill/decode stages, so a latecomer can slot a specialized rack (e.g. decode-only) into an Nvidia-dominated data center. **Alphawave** (acquired) = "Qualcomm's Mellanox" (SerDes/interconnect IP + two heritage hyperscaler custom-silicon customers). **Modular** (acquired, Chris Lattner) = open software layer (Mojo vs CUDA; MAX; multi-silicon token factories). **C1000** data-center CPU (5GHz, 250+ cores) with **Meta** as named customer; $1B revenue target each from two hyperscalers by 2028 ($5B total). Edge AI / AI-defined vehicles / robotics framed as the real long-term prize (claimed $1T by 2040). CEO Amon on memory: "HBM's not going anywhere" + MOAR LPDDR demand at the edge.

## Calls
- **QCOM — NEUTRAL / cautiously constructive:** HBC and edge-AI/robotics vision credible and Qualcomm has the pedigree/IP/cash, but data-center entry is late and HBC is slideware, not shipping silicon; needs a technical paper / Hot Chips proof. Constructive long-term, skeptical near-term. Directly tensions Irrational Analysis's persistent SHORT.
- **META — MENTION (customer):** named C1000 data-center CPU customer with multi-gen agreement.
- **TSM — MENTION:** logic-foundry partner for the HBC accelerator die.
- **NVDA — MENTION:** incumbent Qualcomm is trying to slot beside via disaggregation.
- **Theme — near-memory-compute reinforces the memory bull ("MOAR memory"):** HBC needs *more* DDR wafers, not fewer.
